This repository contains all code needed to reproduce the scRNA-seq and scTCR-seq analyses of the article:

**Antigen-restricted homeostatic Candida albicans Th17 cells link oral-gut immunity and adapt in intestinal inflammation**
 
Gabriela Rios Martini+, Philipp Hofmann+ et al. Antigen-restricted homeostatic Candida albicans Th17 cells link oral-gut immunity and adapt in intestinal inflammation. _Immunity_ 2026. [Online here](https://www.cell.com/immunity/fulltext)
